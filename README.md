# 🦾 RoboDK KUKA Pick-and-Place Simulation

> Industrial pick-and-place simulation for a 6-DOF KUKA robot 
> programmed in RoboDK and validated against real-world execution.

---

## 📋 Overview
Designed and programmed a **pick-and-place robotic sequence** for a 
**6-DOF KUKA industrial robot** using RoboDK simulation software.  
The validated simulation was then executed on a **real KUKA robot** 
in a laboratory environment to identify kinematic deviations and 
optimise path accuracy.

---

## ⚙️ What's Included
- `.rdk` simulation file — open directly in RoboDK
- Full pick-and-place motion sequence
- Real-world validation footage

---

## 🔧 Hardware & Software
- **Robot:** KUKA Industrial Robot (6-DOF)
- **Software:** RoboDK
- **Environment:** THWS Robotics Laboratory
